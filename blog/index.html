Comand prompt and the apocalypse

What the command promt is for?

Well, time ago I was wondering that, and I came to the conclusion that this tool is: the one that few people really know and the one that a lot of people have no idea that it exist

This "terminal" or in common language known as "CMD" by some or "Command promtp" for others, is notthing more than a program, as it doesn't have a graphical interface allow us to manipulate the Operating system in a fast and advance way based on commands. This terminal is divided in two the Terminal itself (interface) and the shell (the core).

In technical words, the terminal is that interface that allow us to interact with the shell, and the shell is the text interface that take the commands and translate it to machine language to say to the computer what it has to do

Advantage

  • Over 9000

    The work flow i really fast, since as it doesn't have a graphical interface, all the commands executes in a direct way with the computer. Compared to a graphical interface, in addition to being pretty, it require more resourses and other commands that runs alongside what we need, as extra services.

  • Danger Zone

    Since as we can work without a graphical interface, the command prompt allow us to communicate with low layers of the operating systems, it can be positive regarding to work flow, but also it can result dangerous, because while we work at low layers of the O.S. we can execute commands that can damage or delete primal files of our system
